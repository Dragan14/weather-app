# Weather App

This is a website showcasing my JavaScript, HTML, and CSS skills. It uses APIs to retrieve location data and weather data.

## Features

-   Responsive design
-   Interactive elements
-   Smooth animations
-   Real-time weather updates

## Technologies Used

-   HTML5
-   CSS3
-   JavaScript (ES6+)
-   OpenWeatherMap API
-   Geolocation API
